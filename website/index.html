What it does

Full location control

📍

Pin Any Location

Click anywhere on the map to instantly teleport your iPhone's GPS to that exact spot. Works globally.

🚗

Drive Simulation

Set a start and destination. GhostLocation calculates a real road route and moves your location along it.

⚡

60–170 mph Speed Slider

Drag the speed slider to control how fast you move along the route. Set it before you start.

📶

WiFi After First USB

Connect once via USB, enable WiFi Sync in Finder, then unplug. Works wirelessly from then on.

🔍

Location Search

Search for any address, city, or landmark and jump straight to it on the map.

🌐

iOS 17+ Support

Uses pymobiledevice3 tunnel for full iOS 17 and later compatibility, including the latest iPhones.

Getting started

Setup guide

1

Install dependencies

Open Terminal and run:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
then brew install libimobiledevice and pip3 install pymobiledevice3

2

Enable Developer Mode on iPhone

Settings → Privacy & Security → Developer Mode → toggle ON. Your iPhone will restart.

3

Connect via USB & Trust

Plug in your iPhone. Tap Trust on the iPhone screen when prompted. Your device should appear in GhostLocation's sidebar.

4

Start the iOS 17+ tunnel

In Terminal run: sudo python3 -m pymobiledevice3 remote tunneld
It will print an --rsd HOST PORT line. Keep this window open and paste those values into GhostLocation.

5

Spoof away

Click the map to drop a pin, or switch to Drive mode and click two points for a route. Hit Start Spoofing.


⚠️

macOS Gatekeeper notice

Because GhostLocation isn't distributed through the App Store, macOS will warn you on first launch. After downloading, right-click the app → Open → Open anyway to bypass Gatekeeper. You only need to do this once.
